# ReconParser

Ferramenta de Reconhecimento (Recon) em Bash para análise de hosts, parsing de HTML e descoberta de hosts online.

## 🔧 Funcionalidades

- Parsing automático de hosts em arquivos HTML
- Remoção de lixo e filtragem de domínios
- Checagem de hosts online
- Suporte a múltiplas threads
- Output organizado

## 🚀 Instalação

```bash
git clone https://github.com/SrKinnG/reconparser.git
cd reconparser
chmod +x recon-parsing.sh
